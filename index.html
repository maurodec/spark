Go-Spark

Simple Bar Graphs for your Terminal

Create simple Sparklines in you Terminal easily!

Go-Spark is a clone written in Go of Holman's Spark shell script.

» seq 0 12 | gshuf | spark
  ▇▃▁▃▆▅▁ ▂▇▅█▄

Go-Spark also features an alternate bar style.

» seq 0 12 | gshuf | spark -alternate
  ⣶⣿⣤⣤⣤⣶⣿⣀ ⣀⣶⣿⣀

It can be used to visualize simple data in your Terminal. Some creative and useful examples can be found in the original Spark's examples page.

Installation

Fetch the source:

go get github.com/maurodec/spark

Install the package

go install github.com/maurodec/spark

Binaries are available in the Releases page!
